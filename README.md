# leekim Publishing House Website

이 프로젝트는 Jekyll 기반의 정적 웹사이트입니다.

## 시작하기 (Getting Started)

이 프로젝트를 로컬 컴퓨터에서 실행하려면 Ruby와 Bundler가 설치되어 있어야 합니다.

### 1. 의존성 설치 (처음 한 번만)

터미널에서 프로젝트 폴더로 이동한 후, 다음 명령어를 입력하여 필요한 라이브러리(Gem)들을 설치합니다.

```bash
bundle install
```

### 2. 로컬 서버 실행

설치가 완료되면, 다음 명령어로 로컬 개발 서버를 실행할 수 있습니다.

```bash
bundle exec jekyll serve
```

또는 코드가 변경될 때마다 화면을 자동으로 새로고침하려면:

```bash
bundle exec jekyll serve --livereload
```

### 3. 웹사이트 접속

서버가 실행되면 브라우저에서 아래 주소로 접속하세요.

http://localhost:4000
