# 도서 데이터 관리 매뉴얼 (LKPH)

이 문서는 `lkph_kr` 사이트의 도서 정보를 수정하거나 삭제하는 방법을 설명합니다.

## 1. 데이터 파일 위치
모든 도서 데이터는 다음 경로의 파일에 저장되어 있습니다:
`_data/books_import_data.tsv`

---

## 2. 파일 구조 이해
이 파일은 **TSV (Tab-Separated Values)** 형식입니다.
*   **구분자**: 열과 열 사이는 쉼표(,)가 아닌 **탭(Tab)** 문자로 구분됩니다.
*   **줄바꿈**: 가독성을 위해 각 도서 정보 위아래로 **빈 줄**이 하나씩 포함되어 있습니다. (1권당 3개 행 차지)
*   **열 구성**: ISBN, 부가기호, 제목, 부제목, 저자, 번역, 가로, 세로, 페이지, 가격, 상태, 발행일, 출판사, 책 소개, 목차, 서평, 저자소개, 발췌문 순서입니다.

---

## 3. 정보 수정하기 (Modification)
1.  `_data/books_import_data.tsv` 파일을 텍스트 에디터(VS Code 등)로 엽니다.
2.  수정할 도서를 검색(Ctrl + F)하여 해당 행을 찾습니다.
3.  원하는 열의 내용을 수정합니다.
    *   **주의**: 글자 사이의 **탭(Tab) 문자가 지워지지 않도록** 주의하세요.
    *   **팁**: 줄바꿈이 필요한 내용(책 소개, 목차 등)은 HTML 태그인 `<br>`을 사용하세요.
4.  파일을 저장하면 사이트가 자동으로 다시 빌드됩니다.

---

## 4. 정보 삭제하기 (Deletion)
1.  삭제할 도서의 정보가 들어있는 행을 찾습니다.
2.  해당 도서의 **데이터 행과 위아래 빈 줄을 포함하여 3개 행을 통째로 삭제**합니다.
3.  파일을 저장하면 목록과 상세 페이지에서 해당 도서가 사라집니다.

---

## 5. 중요 주의사항
*   **따옴표 사용**: 내용 중에 따옴표를 쓸 때는 가능하면 `"`(직선) 대신 **`“”` (둥근 따옴표)**를 사용해 주세요. (TSV 파싱 오류 방지)
*   **인코딩**: 파일을 저장할 때 인코딩은 **UTF-8 (BOM 없음)**이 권장되지만, 시스템에 자동으로 처리하는 기능(`bom_remover.rb`)이 있으니 너무 걱정하지 않으셔도 됩니다.
*   **에이전트 활용**: 직접 수정이 번거롭다면 에이전트에게 다음과 같이 요청하세요:
    > "ISBN 979-11-89680-XX-X 책의 가격을 20,000원으로 수정해줘."
    > "ISBN 979-11-89680-XX-X 도서 정보를 삭제해줘."

---

## 6. 문제 발생 시 확인
수정 후 사이트에 반영이 안 되거나 에러가 난다면 다음을 확인하세요:
*   탭 구분이 깨져서 열의 개수가 달라지지 않았는지 확인합니다.
*   Jekyll 빌드 로그에 에러 메시지가 있는지 확인합니다.
